# OknaKanban

Projekt zaliczeniowy z przedmiotu Programowanie urządzeń mobilnych.

Aplikacja służy do zarządzania zleceniami montażu okien w formie tablicy Kanban. Użytkownik może dodawać, edytować, usuwać oraz zmieniać status zleceń.

## Technologie

- Kotlin
- Jetpack Compose
- Room / SQLite
- Android Studio

## Funkcje

- dodawanie zleceń
- edycja zleceń
- usuwanie zleceń
- podział zleceń na statusy
- zapis lokalny danych
- wyszukiwanie zleceń

## Uruchomienie

Projekt należy otworzyć w Android Studio, poczekać na synchronizację Gradle i uruchomić aplikację na emulatorze lub urządzeniu z Androidem.
